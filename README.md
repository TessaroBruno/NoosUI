<h1 align="center">NoosUI framework</h1>

Noos UI is a C# WinForms custom component library, you can use to create modern appearance app with old WinForms

# Elements

* Forms
* Themes engine
* Localization system
* SplashScreen
* Login page
* Collapsible sidebar
* Form header
* Checkbox
* Combobox
* Buttons
* Datagridview
* Panel
* Label
* Listbox
* Notification
* NumericUpDown
* Switch toggle
* Tab control
* Textbox

## Usage

Launch NoosUIDemo project to run sample app
